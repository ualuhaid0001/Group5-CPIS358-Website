AmanTaxi is a student project prototype for a safe ride website for parents.

The problem

Bus and private taxis are quite a daunting topic to parents who want their kids' safety. Hence AmanTaxi which is advanced and on constant monitor.

Our solution

  • Parent dashboard to request and monitor rides
  • Driver dashboard to see assigned children and trips
  • Child page with large buttons and SOS
  • Admin view to monitor all demo rides
  • Track page with live map and cameras (prototype)
